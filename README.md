# Introduction
The system is built around three STM32F1 microcontroller boards that work together to perform all control and monitoring functions. The distributed architecture improves system flexibility and simplifies future expansion. Inter-board communication is implemented using the UART protocol, ensuring efficient and reliable data transmission between modules.
